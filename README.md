## <a name="projectintro">프로젝트 소개</a>
- 비대면 학습이 증가하면서 PC< 태블릿을 통해 웹으로 학습하는 사례가 늘고 있지만 학습계획을 관리할 수 있는 웹사이트는 적다는 것을 파악하였습니다. 
- 스터디 사이트의 필요성을느끼고 개인 또는 그룹 스터디를 체계적으로 관리할 수 있는 웹 사이트를 제작하였습니다.
